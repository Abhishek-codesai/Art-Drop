## 📌 About

**Art Drop** is a sleek, gallery-style art marketplace where users can browse and "purchase" premium artworks. It's the capstone project of my **Generative AI Summer Training**, built end-to-end through AI-assisted development in Antigravity IDE — proof of applied prompt engineering, not just theory.

## 🛠️ Tech Stack

| Layer | Tech |
|:--|:--|
| 🎨 Frontend | HTML, CSS, JavaScript |
| 🤖 Dev Tool | Google Antigravity IDE (AI-powered) |
| ⚙️ Backend | Google Apps Script |
| 🗄️ Database | Google Sheets (serverless) |
| 🚀 Hosting | GitHub Pages |

## ✨ Features

- 🖼️ Gallery-style artwork showcase, fully responsive
- 🛒 Order capture flow → synced live to a Google Sheets backend
- ⚡ Built entirely through AI-assisted prompting — zero boilerplate written by hand

## 🚀 Quick Start

```bash
git clone https://github.com/abhishek-codesai/Art-Drop.git
cd Art-Drop
```

Open `index.html` in your browser. *(Order submission needs your own Apps Script Web App URL set in `script.js`.)*
